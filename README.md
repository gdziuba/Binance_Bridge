# New Project



chalice new-project <<Project Name>>
cd project
git clone https://github.com/gdziuba/Binance_Bridge.git
pip3 install -r requirements.txt
Change api in app.py
Test Local
chalice local 
chalice deploy

Make sure to add your api keys to the script
